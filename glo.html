Universal Calculator Glossary

Number System Conversion

The process of converting numbers between different bases, such as:

  • Decimal (Base-10): Standard system using digits 0–9.
  • Binary (Base-2): System using digits 0 and 1.
  • Hexadecimal (Base-16): System using digits 0–9 and letters A–F.
  • Octal (Base-8): System using digits 0–7.

Bitwise Operations

  • AND: Returns 1 if both bits are 1.
  • OR: Returns 1 if at least one bit is 1.
  • XOR: Returns 1 if bits are different.
  • NOT: Inverts all the bits.
  • Left Shift: Shifts bits to the left, filling with zeros.
  • Right Shift: Shifts bits to the right.

Base64 Encoding

A binary-to-text encoding scheme that converts binary data into a sequence of printable characters, commonly used for encoding binary data in text formats like emails, URLs, or data URIs.

ASCII (American Standard Code for Information Interchange)

A character encoding standard that represents text in computers and other devices by assigning numeric values to letters, digits, punctuation marks, and control characters.

Morse Code

A method of encoding text characters as sequences of dots and dashes (or dits and dahs), historically used in telegraphy, and still used in certain emergency communications.
