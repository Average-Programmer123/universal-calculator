üîÑ Convert Decimal to Binary

Converting decimal to binary involves dividing the number by 2 and tracking the remainders. These remainders, when read in reverse, give the binary equivalent.

This method is known as successive division and is a fundamental concept in computer science education.

1 √∑ 2 = 0 remainder 1

üß† Bitwise & Bit Shift Operations

Bitwise operations are crucial in programming for performance optimization, device control, and encryption. These operations manipulate bits individually using binary logic.

  • AND (&): True only if both bits are 1.
  • OR (|): True if either or both bits are 1.
  • XOR (^): True if the bits are different.
  • NOT (~): Inverts all bits (1 ‚Üî 0).
5 = 0101
3 = 0011
5 & 3 = 0001 = 1
5 | 3 = 0111 = 7
5 ^ 3 = 0110 = 6
  • Use of Bitwise Operations:
    • Optimizing performance in low-level programming (e.g., C, C++).
    • Controlling hardware registers and device drivers.
    • Encryption, compression, and data masking techniques.

üîÄ Bit Shifting

Shifting bits to the left or right moves them by a certain number of positions. This can be used for efficient multiplication or division by powers of two.

  • Left Shift (<<): Shifts bits left, filling with 0. Equivalent to multiplying by 2 for each shift.
  • Right Shift (>>): Shifts bits right, discarding bits on the right. Equivalent to integer division by 2 for each shift.
6 = 0110
6 << 1 = 1100 = 12
6 >> 1 = 0011 = 3
  • Bit Shifting:
    • Fast multiplication or division by powers of two.
    • Efficient image processing and mathematical algorithms.
    • Microcontroller and embedded system operations.

üî§ Text and ASCII Operations

ASCII (American Standard Code for Information Interchange) assigns numeric values to characters. Computers use these values to store and manipulate text.

  • Character to ASCII: Converts each letter or symbol to its corresponding ASCII code.
  • ASCII to Character: Converts a numeric ASCII value back into a readable character.
  • Text to Binary: Shows how characters are represented in binary format.
  • Reverse Text: Reverses the string letter by letter.
  • Reverse ASCII: Flips the ASCII codes rather than characters themselves.
Text: Hi
ASCII: H = 72, i = 105
Binary: H = 01001000, i = 01101001
Reversed: iH
  • Use of ASCII & Text Operations:
    • Text encoding in programming and data transmission.
    • Web development and terminal communications.
    • String manipulation and input validation in software development.

    • üßÆ Binary Arithmetic

    Just like regular math, binary addition and subtraction follow logical rules. Carrying and borrowing work similarly to the decimal system, but only with 0s and 1s. It's more efficient to first convert the binary to decimal and then andd the terms and convert the final number to binary.

    • Addition: Add bits from right to left, carrying when needed.
    • Subtraction: Subtract bits with borrowing if the top bit is smaller than the bottom bit.
    1011 (11 in decimal) + 0101 (5 in decimal) = 10000 (16) = 16 in decimal
    • Use of Binary Arithmetic:
      • Core to processor design and CPU logic operations.
      • Arithmetic circuits in digital calculators and computing devices.
      • Used in error detection algorithms and data encoding.

    üîê Encoding & Decoding

    Encoding transforms information into different formats for safe transmission or storage. Decoding reverses that transformation to restore the original message.

    • Base64 Encode/Decode: Encodes binary data as text. Common in web APIs and emails.
    • URL Encode/Decode: Safely sends data in URLs by converting special characters.
    • HTML Entities: Prevent code injection and formatting errors in HTML.
    • Hex Encode: Converts text into hexadecimal, useful in debugging and cryptography.
    • Morse Code: Encodes text into dot-dash signals used in radio communication.
    • ROT13: A simple letter substitution cipher that shifts letters 13 places.
    Text: Hello
    Base64: SGVsbG8=
    URL Encoded: Hello%20World
    Morse: .... . .-.. .-.. ---
    • Encoding & Decoding (Base64, URL, etc.):
      • Base64: Used in email attachments and web APIs.
      • URL encoding: Safely sending text in links and web forms.
      • Morse code: Emergency communication and amateur radio.
      • ROT13: Simple data obfuscation (e.g., in puzzles and forums).
      • HTML entities: Preventing code injection and formatting issues in web pages.
